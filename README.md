# colorful-chat-ai-tool

A colourful and playful chat with Image (Image to text) AI Tool which allows users to upload images of their choice and analyses the image and facilitates a chat conversation with the user regarding the image

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/colorful-chat-ai-tool.git
cd colorful-chat-ai-tool
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
